# 本轮转化优化实施记录

> 日期：2026-04-24

---

## 一、这轮做了什么

修复"内容→服务"转化链路的 5 个断点，让文章读者、资料库用户、About 页访客、Contact 页访客都能找到通向服务页的路径。

---

## 二、修改了哪些页面

| 页面文件 | 改动内容 |
|----------|---------|
| `app/blog/[slug]/page.tsx` | 文章末尾 CTA 改为按分类显示（上一轮已完成） |
| `app/page.tsx` | 首页 Hero 增加低调服务入口（上一轮已完成） |
| `app/services/page.tsx` | 每个服务卡片增加「先了解判断逻辑」相关文章（上一轮已完成） |
| `app/resources/page.tsx` | 装修类资料（报价审核清单、预算模板、实住派自查表）底部增加服务 cross-sell |
| `app/about/page.tsx` | 末尾增加「如果你需要更具体的判断」服务出口区块 |
| `app/contact/page.tsx` | 顶部增加意图分流区块（服务/资料库/直接联系三路分流） |

---

## 三、每个页面加了什么入口

### 资料库页 `/resources`
- 报价审核清单 → "审核服务会比清单更直接" → /services
- 预算模板 → "具体的判断比再看一份模板更有效" → /services
- 实住派自查表 → "先用资料建立判断，再决定是否需要服务" → /services

### About 页 `/about`
- 新增区块：「如果你需要的不是更多信息，而是更具体的判断」
- 提到装修和 AI 两个场景
- 一个按钮：查看服务 → /services

### Contact 页 `/contact`
- 顶部新增三列分流卡片：
  - 有明确问题 → 查看服务
  - 还在了解 → 先看资料库
  - 合作/其他 → 往下看联系方式

---

## 四、这轮不做什么

- 不改首页整体视觉
- 不改英文站
- 不新增支付/会员
- 不做复杂埋点
- 不自动群发公众号
- 不重构全站
