## 一、接下来 2-4 周重点看什么

### 1. 哪篇文章带来更多咨询

去 Vercel Analytics → Top Pages，看 `/blog/[slug]` 的访问量排名。
再看 `/services` 页面的 referrer（从哪来的），如果某篇文章名频繁出现，说明那篇文章在做"文章→服务"的转化。

重点关注这几篇（预期最有转化潜力）：
- `/blog/zhuangxiu-yusuan-weishenme-zongchao`（预算超支）
- `/blog/baojia-dan-zenme-kan`（报价单怎么看）
- `/blog/article-05-01`（传统行业学AI）
- `/blog/neirong-xitong-jianqi`（内容系统怎么建）

### 2. 哪个 CTA 最有效

目前无法直接看按钮点击率（需要埋点或自定义事件，暂未接入）。
替代方法：看 `/services` 的访问量是否在改动后有明显提升。
如果提升了，说明新增的 CTA 入口在起作用。

### 3. 哪个服务入口点击率最高

看 `/services` 页面的来源路径分布：
- 从 `/blog/[slug]` 来的 → 文章 CTA 有效
- 从 `/resources` 来的 → 资料库 cross-sell 有效
- 从 `/about` 来的 → About 服务出口有效
- 从 `/contact` 来的 → 联系页分流有效
- 从 `/`（首页）来的 → 首页 Hero 链接有效

### 4. 资料库是否在把人送向服务

看 `/resources` 页的下一步去向：
- 去了 `/services` → cross-sell 在工作
- 去了 `/contact` → 用户在走领取流程（正常）
- 去了 `/blog` → 用户在回流看文章（也正常，但说明资料库不是终点）

### 5. About 页新增出口有没有人用

看 `/about` 的下一步点击。如果 `/services` 出现在后续路径中，说明有效。

### 6. Contact 页分流效果

看 `/contact` 的下一步去向：
- 去了 `/services` → 分流区块有效，用户意识到自己应该先看服务
- 去了 `/resources` → 分流在起作用，把初步了解的人送去了正确的地方

## 二、重点观察页面

| 页面 | 看什么 |
|------|--------|
| `/blog/[slug]` | 访问量、停留时间、下一步去了哪（是否去了 /services） |
| `/resources` | 访问量、下一步去向（是否有人从资料页去了服务页） |
| `/about` | 下一步去向（是否有人点了服务出口） |
| `/contact` | 下一步去向（分流区块是否在引导用户去服务页或资料库） |
| `/services` | 访问量变化趋势、来源分布 |

---

## 三、每个页面看什么指标

用最简单的话说：

- **访问量**：多少人来了这个页面
- **停留时间**：来了之后待了多久（文章页停留时间越长越好）
- **下一步点击**：看完这个页面，用户去了哪里
- **是否点进服务页**：这是当前最重要的转化指标
- **是否从服务页继续联系**：看 `/services` 之后有没有人去 `/contact`

## 四、怎么判断这轮改动有没有价值

### 有效的信号：
- `/services` 页面周访问量比改动前增加 30% 以上
- `/services` 的来源路径里出现了 `/blog/[slug]`、`/resources`、`/about`（之前几乎没有）
- 微信上开始有人备注"报价审核"或"AI咨询"（说明 CTA 文案被读到了）

### 无效的信号：
- `/services` 访问量没有变化
- 来源路径仍然几乎全是直接访问或首页
- 没有新增咨询消息

### 复盘时间：
改动上线 **2 周后**做第一次数据查看。
**4 周后**做正式复盘，决定是否需要调整 CTA 文案或增加新入口。
