## 英文站"不做什么"清单

| 不做的事 | 原因 |
|----------|------|
| 资料库（中文 PDF） | 英文用户看不了中文资料 |
| 公众号二维码 | 国际用户不用微信 |
| 50 篇全量文章 | 英文宁少勿滥 |
| 南宁本地内容 | 对国际读者无意义 |
| 报价审核服务 | 中文业主专属 |
| 中文排版工具说明 | 放入 Tools 即可，不需要独立导航位 |
